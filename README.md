<h1 align="center">SMART-T</h1>

<p align="center"><strong>SMART-T (Sumo-based Mobile Autonomous Robot w/ Tank-Treads) implements simultaneous localization and mapping conepts as well as sumo functionality on the Pololu Zumo 32U4 platform. It is intended to compete in robotic sumo events</strong></p><br/><br/>

<div align="center">

# 3 Point Mapping Technique

<p>SMART-T uses a mapping technique that derrives the location of points from preceeding ones. After 3 points have been identified, SMART-T can calculate the location of a circle's center and move to that point.</p>

![Mapping Visual](https://github.com/g-caldwell/SMART-T/blob/main/README%20Assets/mappingVisual.png)
<br/><br/>

# Centerpoint Formula

<p>The algebraic linearization of the general equation for a circle, as well as the formula for finding the radius given the values of the center's location.</p>
