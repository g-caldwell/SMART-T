<h1 align="center">SMART-T</h1>

SMART-T (Sumo-based Mobile Autonomous Robot w/ Tank-Treads) implements simultaneous localization and mapping conepts as well as sumo functionality on the Pololu Zumo 32U4 platform. It is intended to compete in robotic sumo events.
